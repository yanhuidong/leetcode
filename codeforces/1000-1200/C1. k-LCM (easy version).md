# [C1. k-LCM (easy version)](https://codeforces.com/problemset/problem/1497/C1)

这道题虽然是eazy版的，但还是有一定的学习意义的，我也成功靠自己的努力自己想出了正确的思路，下面是我的代码：

```cpp

#include<bits/stdc++.h>
using namespace std;

int t, n, k;

void solve() {
    cin >> n >> k;
    k = 3;
    int half = n / 2;
    if (n % 2 == 1) {
        cout << half << ' ' << half << ' ' << 1 << endl;
    }

    else {
        if (n % 4 == 0) {
            int halff = half / 2;
            cout << half << ' ' << halff << ' ' << halff << endl;
        }

        else {
            cout << half - 1 << ' ' << half - 1 << ' ' << 2 << endl;
        }
    }
}

int main() {
    cin >> t;
    while (t--) {
        solve(); 
    }
    return 0;
}
```
